Current user name:
```bash
whoami
```

Current user UID, GID and groups:
```bash
id
```

Show which commands we are can run as sudo:
```bash
sudo -l
```

Current user history:
```bash
history
```

Show system users:
```bash
cat /etc/passwd
```

Can we read user password hashes?:
```bash
cat /etc/shadow
```

Can we read group file?:
```bash
cat /etc/group
```
